🙋 What is this?

This is a single-page application, which means this single page contains everything for this app.

Features:

  • Generates standard meta tags based on Open Graph (Facebook) and Schema.org.
  • This stores your last generated form (in localStorage) so when you get back to it, it remembers what you entered last.
  • Yes this is 👨‍💻 Open Source!

🤷 Why?

I haven't found a good enough tool that would do this basic thing as generate text for my meta tags.

To speed up the process of building small apps, I built this. I also saw that other tools available are unnecessarily complicated.
