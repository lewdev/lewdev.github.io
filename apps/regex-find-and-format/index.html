Regex Find & Format Tool

This app was built to help me perform data scraping-like operations and formatting those results. I intially did this on Notepad++ using its "find and replace tool" with regular expressions, but to get what I find and then put those results into a format require extra steps, so I turned the process into a tool.

Why?

I am publishing this tool because I figured that it might be useful to somebody else. I have seen similar tools, but they don't have the format component that I find most valuable.
